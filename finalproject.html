Final Project

The purpose of “Letter Tracing for Fine Motor Control” is to help children with fine motor difficulties improve their handwriting skills.
The program is designed to be used on a touch screen device, such as an iPad, and with a stylus in order to mimic learning to write the
alphabet. The letters are capital, as these are easier for children to learn than lowercase letters. The program will allow professionals
to quantify the quality of children’s handwriting and be used as a way to compare results over time.
